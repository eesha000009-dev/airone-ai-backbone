# Airone AI Backbone

Desktop application for robot AI control with Triune Brain architecture.

## Features
- Robot Identity Management (create, configure, and manage robots)
- AI Model Selection (OpenAI, Claude, Ollama, or Rule-based)
- Hardware Map (visual pin mapping and configuration)
- Live Monitor (real-time sensor data and AI commands)
- Built-in Brain Server (WebSocket on port 8080)
- Triune Brain Architecture (Reflex → Memory → Goal)
- SQLite database for persistence

- No Python required (standalone Electron app)
